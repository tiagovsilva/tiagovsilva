## Open-Source Automation

The [PowerShell Infrastructure Automation](powershell-infrastructure-automation/) collection contains documented and sanitized scripts for:

- Microsoft Entra ID and Microsoft Graph
- Microsoft 365, Exchange Online, and SharePoint Online
- Active Directory and hybrid identity
- Azure File Sync, WSUS, VMware, and Windows administration

Scripts are separated by `cloud`, `hybrid`, and `onpremise`, with explicit PowerShell 5.1 or 7.4 requirements.

> **Production safety:** Review every script and test it first in a controlled, non-production environment. Validate the result, impact, failure behavior, and rollback path before considering production execution.
