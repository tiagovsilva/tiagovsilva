# Cloud Automation

Cloud scripts use current service modules where possible and are grouped by platform:

- [Azure Files](azure-files/) - Azure File Sync agent configuration.
- [Microsoft Entra ID](entra-id/) - Microsoft Graph identity and Intune operations.
- [Microsoft 365](microsoft-365/) - Exchange Online and SharePoint Online.
- [Google Workspace](google/) - reserved for future reviewed scripts.

Most Microsoft cloud scripts require PowerShell 7.4. Check each script's `#Requires` declarations.
