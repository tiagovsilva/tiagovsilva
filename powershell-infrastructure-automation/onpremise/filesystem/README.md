# Filesystem

| Script | Purpose |
|---|---|
| [`invoke-robocopy.ps1`](invoke-robocopy.ps1) | Copies or mirrors a directory tree with logging. |
| [`new-directory-symbolic-link.ps1`](new-directory-symbolic-link.ps1) | Creates a directory symbolic link. |

Mirror mode can delete destination-only files. Use `-WhatIf`.
