# Code of Conduct

## Our standards

Participation in this repository must remain respectful, technical, and constructive.

Examples of acceptable behavior include:

- Providing reproducible technical context
- Reviewing code and ideas without attacking people
- Respecting privacy, confidentiality, and intellectual property
- Accepting correction when a security or reliability concern is demonstrated

Unacceptable behavior includes harassment, discrimination, personal attacks, publishing private information, or deliberately submitting secrets and proprietary data.

## Enforcement

Report conduct concerns privately to the repository owner through the public profile. Repository maintainers may edit, remove, or reject contributions and may temporarily or permanently restrict participation when necessary.
